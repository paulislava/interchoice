Несколько простых шагов

Как создается интерактивный видеоконтент?

  1. 1. Выберете в Меню пользователя пункт “Создать фильм”.
  2. 2. Придумайте название для Вашего фильма и загрузите эффектное изображение в качестве заставки.
  3. 3. Создавайте необходимое количество сцен, для Вашего кино, загружая заранее подготовленное видео.
  4. 4. Чтобы просмотреть созданный фильм нажмите “Просмотр фильма” в Меню.
  5. 5. Когда всё будет готово, нажмите “Опубликовать фильм” и ваше кино появляется в Ленте.
